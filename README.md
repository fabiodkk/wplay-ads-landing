# WPlay Ads Landing

Pagina estatica para divulgacao do teste WPlay em anuncios.

## Arquivos principais

- `index.html`: destino principal do anuncio.
- `termos.html`: termos de uso.
- `privacidade.html`: politica de privacidade.
- `styles.css`: estilos responsivos.
- `script.js`: aquecimento best-effort da area tecnica externa.

## Observacao

Use esta pagina apenas se a oferta, os direitos de conteudo, o atendimento e a categoria anunciada estiverem de acordo com as politicas do Google Ads e com a legislacao aplicavel.
